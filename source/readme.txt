Light PHP Edit for PHP 5.1.2.2
Licence GPL 2 read COPYING.TXT file

Yet another php editor, Maked as light and easy and simple, integrated with PHP by php4delphi component,
Highlight the code useing SynEdit component, has internal console for learning php easy.

Site: http://www.parmaja.com/projects/LightPHPEdit

To work with php you must install/extract the zip file in c:\php or where you want, then point to php folder by open setting Menu>Tools>Setting
And point to CHM help file after install it from php site.
